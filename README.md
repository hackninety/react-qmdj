# 奇门遁甲排盘系统 (React QMDJ)

基于 React + Vite 打造的现代化、跨平台奇门遁甲排盘与分析系统。内置高精度历法计算与真太阳时校准，同时完美融合大语言模型（LLM）实现 AI 自动化断局。

## ✨ 核心特性

- **精准排盘算法与传统流派**
  基于底层的 `lunar-javascript` 高精度天文历法数据，支持完整的**时家、日家、月家、年家**奇门排盘。
  - **排盘流派**：采用正统的**转盘奇门**排布法（非飞盘），天盘沿洛书八宫整体旋转。
  - **定局方法**：以节气定阴阳遁，以日干支地支（四仲/四孟/四季）定上中下元，查表得局数。属于**直接定局法**，在多数场景下与拆补法结果一致。
  - **中宫寄宫**：遵循经典的**中宫寄坤二宫**规则（天禽星与第五宫的天盘干随坤二宫同转）。
  
- **真太阳时与时区校准**
  无论你身处哪个国家或时区：
  - **默认模式**：直接使用本机时间快速起盘。
  - **自定义排盘**：支持按城市地区选择经纬度计算。系统会自动将时间归一化为中国标准时间（UTC+8），并叠加该经度的**真太阳时修正**（包含了经度偏差与地球公转时差方程），确保时辰交接与节气点绝对精准。

- **多维断局信息展示**
  不仅提供直观的天地盘九宫格（含九星、八门、八神、干支、暗干、隐星等），更在下方“九宫详解”面板中提供了每个宫位的吉凶、格局含义、具体意象分析。

- **AI 大模型零代码对接**
  - **JSON 导出**：将复杂的排盘结果一键转化为结构化的 JSON 数据。
  - **内置 Prompt 模板**：自带为奇门量身定制的 AI 提示词库（包含日时相冲、旬空马星、九宫生克、用神定位等 7 大专业分析维度）。
  - 点击“复制 AI Prompt”即可丢给 ChatGPT / Claude / Gemini 进行深度万物断局，彻底解放人力。

- **现代化 UI 体验**
  - **玻璃拟态美学**：精致的卡片式设计与顺滑的 `framer-motion` 动效。
  - **单列响应式布局**：自适应手机端与桌面端，自上而下的阅读流与主流阅读习惯完美契合。
  - 支持浅色（Light）与深色（Dark）模式无缝切换。

## 🚀 本地开发指南

项目采用 Node.js + React + Vite + Tailwind CSS 技术栈。

```bash
# 1. 克隆代码仓库
git clone https://github.com/hackninety/react-qmdj.git

# 2. 进入项目目录
cd react-qmdj

# 3. 安装依赖包
npm install

# 4. 启动本地开发服务器
npm run dev
```

启动后，访问浏览器中提示的 `localhost` 地址（通常是 `http://localhost:6677/`），即可预览和调试系统。

## ☁️ 部署到 Cloudflare Pages (推荐)

此项目包含静态构建特性，极其适合一键部署至 Cloudflare Pages。过程完全免费且自带全球 CDN 加速。

### 部署步骤：

1. 登录到 [Cloudflare 控制台](https://dash.cloudflare.com/)。
2. 在左侧导航栏选择 **“Workers 和 Pages”**。
3. 点击 **“创建应用程序”**，切换到 **“Pages”** 选项卡，然后选择 **“连接到 Git”**。
4. 授权并选择你的 `react-qmdj` GitHub 仓库。
5. 在**设置构建和部署**页面，填写如下信息：
   - **框架预设 (Framework preset)**：选择 **`None`** 或者 **`Vite`**
   - **构建命令 (Build command)**：`npm run build`
   - **构建输出目录 (Build output directory)**：`dist`
6. （非必须但推荐）如果需要在构建时锁定 Node 版本，可以展开“环境变量”，添加 `NODE_VERSION` 变量，其值为 `18`（或更高版本的 LTS）。
7. 点击 **“保存并部署”**。

等待几十秒钟构建完成后，Cloudflare 会分配给你一个 `*.pages.dev` 的在线访问域名。现在你可以不受地域限制随时随地起盘了！
